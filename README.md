# AI-Ml-Project
Weather Prediction System (Without Machine Learning Libraries)
This project is a Weather Prediction System built using Python without using any machine learning libraries like sklearn. It uses basic logic and mathematical calculations to predict weather conditions.

Features
No external ML libraries used
Simple and easy to understand logic
Uses CSV dataset
Takes user input for prediction
Based on distance calculation method

How It Works
The dataset is loaded using Python's CSV module.
Data is separated into two categories:
Sunny
Rainy
Average values for each category are calculated.
The system compares user input with averages using distance formula.
The closest match determines the predicted weather.
How to Run
python weather_prediction.py
Example
Enter Temperature: 29
Enter Humidity: 70
Enter Wind Speed: 6

Predicted Weather: Sunny
